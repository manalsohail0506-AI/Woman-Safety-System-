# Smart Woman Safety System

## Description
The Smart Woman Safety System is a Python-based application developed using Object-Oriented Programming (OOP). It helps women during emergencies by providing SOS alerts, medical emergency assistance, emergency contact management, simulated location tracking, incident history, and emotional support.

## Features

- User Registration
- User Login
- SOS Emergency Alert
- Medical Emergency Alert
- Emergency Contact Management
- Incident History
- Simulated GPS Location
- Voice Alert System
- Emotional Support Messages

## OOP Concepts Used

- Classes & Objects
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction

## Technologies

- Python
- File Handling
- OOP
- Abstract Base Classes
